## Librerie
### NITHlibrary
_Main core library for the NITH framework. Multi-platform. Manages data parsing from NITH sensors and wrappers, as well as preprocessing, filtering, and other data manipulations._
Repository: https://github.com/LIMUNIMI/NITHlibrary
### NITHemulation
_Additional section of the library for basic I/O emulation (keyboard and mouse). Available only for Windows operating systems, since it's system-call dependent._
Repository: https://github.com/LIMUNIMI/NITHemulation
### NITHdmis
_Additional section of the library for Accessible Digital Musical Instruments development. Contains basic music manipulation tools._
Repository: https://github.com/LIMUNIMI/NITHdmis

## NITHtester
_A GUI application for testing NITH sensors and wrappers, which provides gauges, an head tracking tester, etc._
Repository: https://github.com/LIMUNIMI/NITHtester

## NITHphoneWrapper
_A NITHwrapper which turns any Android smartphone into a sensor (accelerometer, gyroscope, compass, touchscreen)._
Repository: https://github.com/LIMUNIMI/NITHphoneWrapper

## NITHsensors-firmwares
_A collection of NITHsensors firmwares (mostly Arduino-related)._
Repository: https://github.com/LIMUNIMI/NITHsensors-Firmwares

## NITHwebcamWrapper
_A Python script NITHwrapper which turns any laptop webcam into a sensor (detects head rotation, mouth aperture, eye blinking and winking)._
Repository: https://github.com/LIMUNIMI/NITHwebcamWrapper

## NITHtemplate
_A Visual Studio template for the development of new NITH based applications._
Repository: https://github.com/LIMUNIMI/NITHtemplate

## NITHmouseController
_A NITH application for mouse emulation through head rotation, mouth aperture, eye winking._
Repository: https://github.com/LIMUNIMI/NITHmouseController

## NITHexample
_A simple but complete of NITH application._
Repository: https://github.com/LIMUNIMI/NITHexample
